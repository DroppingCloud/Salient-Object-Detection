\section{摘要}

本报告围绕显著性目标检测任务展开，以 ResNet-18 \cite{he2016deep} 为骨干网络，
系统研究了基于深度学习的显著图预测方法。

\bigskip
\noindent\textbf{关键词：}显著性目标检测；ResNet-18；编解码器；注意力机制；多尺度融合

%----------------------------------------------------------------
\section{引言}

显著性目标检测（Salient Object Detection, SOD）旨在从自然图像中定位并分割出最吸引人类视觉注意力的前景区域，
输出与输入图像等尺寸的二值或软性预测掩码（mask）。该任务在图像检索、视频理解、自动驾驶等领域具有广泛的应用价值。

传统方法主要依赖于在手工设计的特征下进行，难以适应复杂图像场景和结构，泛化能力有限。
随着深度学习的发展，基于卷积神经网络的方法在语义特征提取上取得了显著进步，成为当前的主流范式。

我们的贡献总结如下：

\begin{enumerate}
  \item 实现了两种基于 ResNet-18 的基线方法（FCN 解码器与 U-Net 跳跃连接解码器）并进行对比分析；
  \item 提出了融合边缘增强分支、通道注意力与多尺度特征融合的改进网络（ResSOD-Improved）；
  \item 设计了结合二值交叉熵（BCE）与结构相似性（SSIM）的混合损失函数；
  \item 在 DUTS 数据集上进行了系统的消融实验，定量验证了各模块的贡献。
\end{enumerate}

\subsection{ResNet}

%----------------------------------------------------------------
\subsubsection{基本原理}

ResNet\cite{he2016deep} 是深度卷积神经网络中的经典骨干结构，其核心思想是通过残差连接缓解深层网络训练中的退化问题。
与传统卷积网络直接学习目标映射 $H(x)$ 不同，残差网络学习的是输入与输出之间的残差部分 $F(x)=H(x)-x$。这种设计使得网络在加深时更
容易学习恒等映射，从而降低深层网络的优化难度。

残差块（Residual Block）是 ResNet 的基本组成单元（图 \ref{fig:residual_block}），主要由残差分支和捷径分支两部分构成。残差分支用于对输入特征进行非线性变换，捷径分支则将输入特征直接传递到输出端，并与残差分支的输出进行逐元素相加。

\begin{figure}[t]
  \centering
  \includegraphics[width=0.7\textwidth]{fig/resnet1.png}
  \caption{Residual block}
  \label{fig:residual_block}
\end{figure}

%----------------------------------------------------------------
\subsubsection{网络结构}

ResNet 通常由一个初始卷积层和若干个残差阶段组成（图~\ref{fig:resnet}）。以 ImageNet 版 ResNet 为例，
输入图像先经过 7×7 conv + BN + ReLU + maxpool 得到低层特征，然后依次经历 layer1、layer2、layer3、layer4 四个残差阶段。

每个阶段由多个残差块组成，空间分辨率逐步下降，通道数逐步增加。一般情况下，ResNet18/34 使用 BasicBlock，ResNet50/101 使用更深的 Bottleneck Block。

\begin{figure}[t]
  \centering
  \includegraphics[width=0.7\textwidth]{fig/resnet3.png}
  \caption{ResNet 网络结构}
  \label{fig:resnet}
\end{figure}

%----------------------------------------------------------------
\subsubsection{基于 ResNet 的显著目标检测网络}

显著目标检测（Salient Object Detection, SOD）旨在从图像中预测最具视觉吸引力的前景区域。由于该任务需要生成像素级显著图，因此网络既需要高层语义信息来定位显著目标，又需要低层空间信息来恢复目标边界。

基于 ResNet 的显著目标检测网络通常采用编码器-解码器（Encoder-Decoder）结构。其中 ResNet 作为编码器提取多尺度特征，
解码器负责融合不同层级特征并生成显著图。

\begin{align*}
    S = D(C_1, C_2, C_3, C_4)
\end{align*}

其中 $C1, C2, C3, C4$ 表示解码器输出的各层级特征，$D(\cdot)$ 表示解码器，$S$ 表示输出显著图。

% ----------------------------------------------------------------
\subsection{PoolNet}

% ----------------------------------------------------------------
\subsubsection{基本思想}

PoolNet \cite{liu2019simple}是一种基于编码器—解码器结构的显著目标检测网络（图~\ref{fig:poolnet}），其核心思想是充分融合深层语义信息与浅层空间细节信息，从而生成区域完整、边界清晰的显著性预测图。
PoolNet 的核心创新点包括：

\begin{itemize}
    \item \textbf{全局上下文增强}：通过顶层特征的多尺度池化，使网络感受野扩大，可区分局部高对比度背景与真正显著目标。
    \item \textbf{多尺度特征融合}：通过自顶向下的解码过程将高层语义信息与浅层空间信息融合，增强浅层特征的表达能力并抑制噪声。
\end{itemize}

\begin{figure}[t]
  \centering
  \includegraphics[width=1.0\textwidth]{fig/poolnet.png}
  \caption{PoolNet 模型架构}
  \label{fig:poolnet}
\end{figure}

% ----------------------------------------------------------------
\subsubsection{金字塔池化模块（PPM）}

PoolNet 引入金字塔池化模块（Pyramid Pooling Module，PPM）用于捕获多尺度上下文信息。
对于深层输入特征 \( X_{\text{deep}} \in \mathbb{R}^{B \times C \times H/32 \times W/32} \) ，首先经过多尺度自适应平均池化形成多尺度全局上下文特征：
\[
X_{s} = \text{Upsample}\Big(\text{AvgPool}_{s}(X_{\text{deep}})\Big)
\]
其中 \(s \in \{1,3,5\}\) 表示不同池化输出尺寸，Upsample 将池化特征上采样回深层特征分辨率。然后再与原始深层特征进行拼接增强：
\[
X_{\text{PPM}} = \text{Conv}\Big([X_{\text{deep}}, X_{1}, X_{3}, X_{5}]\Big)
\]

PPM 输出随后上采样到浅层特征尺寸，用作 Global Guiding Flow (GGF)，为解码器提供全局语义指导。

% ----------------------------------------------------------------
\subsubsection{深度池化特征融合（Deep Pooling Layer）}

在解码阶段，PoolNet 采用自顶向下融合的策略，核心模块是深度池化层（Deep Pooling Layer）。其处理流程为：

\begin{enumerate}
    \item 高层特征 \(F_{\text{high}}\) 上采样至当前层特征分辨率，
    并与当前层特征 \(F_{\text{skip}}\) 及 GGF 全局信号特征 \(F_{\text{info}}\) 相加。
    \[
    F_{\text{merge}} = F_{\text{high}}^{\uparrow} + F_{\text{skip}} + F_{\text{info}}
    \]
    
    \item 对融合后的特征执行多尺度平均池化（Adaptive Average Pooling），最后每个池化分支通过卷积和 ReLU 后上采样回原始分辨率，
    融合不同尺度的上下文信息，从而增强感受野。
    \[
    F_{\text{pool}} = F_{\text{merge}} + \sum_{s \in \{2,4,8\}} \text{Upsample}\Big(\text{AvgPool}_{s}(F_{\text{merge}})\Big)
    \]
    
    \item 对融合后的特征进行卷积整合。
    \[
    F_{\text{out}} = F_{3\times3}(F_{\text{pool}})
    \]
\end{enumerate}

这种自顶向下的融合策略保证了浅层特征同时拥有高分辨率空间信息和深层语义信息，
提升显著图对边缘和细节的敏感性，同时抑制浅层噪声干扰。

% ----------------------------------------------------------------
\subsection{GateNet}

% ----------------------------------------------------------------
\subsubsection{基本思想}

GateNet \cite{zhao2020suppress} 是一种基于门控机制的显著目标检测网络（图 \ref{fig:gatenet}），
核心思想是在编码器与解码器之间引入门控单元，自适应控制不同层级特征向解码器传递的信息量。
GateNet 的核心创新点包括：

\begin{itemize}
    \item \textbf{特征筛选}：通过门控权重抑制非显著区域响应，减少背景干扰。
    \item \textbf{层级平衡}：根据不同编码层对预测结果的贡献，动态调整特征融合强度。
    \item \textbf{双分支互补}：FPN 分支负责主体区域预测，Parallel 分支补充边界和细节信息。
    \item \textbf{多尺度增强}：利用 Fold-ASPP 增强深层特征的上下文表达能力。
\end{itemize}

\begin{figure}[t]
  \centering
  \includegraphics[width=1.0\textwidth]{fig/gatenet.png}
  \caption{GateNet 模型架构}
  \label{fig:gatenet}
\end{figure}

% ----------------------------------------------------------------
\subsubsection{Fold-ASPP}

对于深层语义特征，GateNet 引入 Fold-ASPP 模块以增强多尺度上下文建模能力。普通 ASPP 通过不同空洞率的卷积扩大感受野，但较大的空洞率会导致采样点之间相关性减弱，不利于细节结构建模。
而 Fold-ASPP 通过 Fold 操作将局部邻域折叠到通道维度，使空洞卷积作用于局部区域而非孤立采样点，从而在扩大感受野的同时保留局部结构连续性（图~\ref{fig:fold}）。

Fold-ASPP 采用多条并行分支提取不同尺度的上下文信息，多尺度分支输出融合后，为后续解码器提供更强的高级语义指导。

\begin{figure}[t]
  \centering
  \includegraphics[width=0.7\textwidth]{fig/fold-aspp.png}
  \caption{普通空洞卷积与折叠空洞卷积对比}
  \label{fig:fold}
\end{figure}

% ----------------------------------------------------------------
\subsubsection{Gate Units}

传统 FPN 或 U-Net 通常直接将编码器特征传递给解码器，但这种 all-pass skip connection 可能将背景噪声和误导性上下文一并引入解码过程。
为此，GateNet 设置门控单元（Gate Units）显式控制编码器特征向解码器传递的信息量。

对于第 \(i\) 层，门控单元同时接收当前编码器特征 \(E_i\) 和上一层解码器特征 \(D_{i+1}\) 来生成门控权重：
\[
G_i =
P\Big(
\sigma(
\text{Conv}([E_i, D_{i+1}])
)
\Big)
\]
其中 \([\,\cdot\,]\) 表示通道拼接，\(\sigma(\cdot)\) 为 Sigmoid 函数，\(P(\cdot)\) 表示全局平均池化。

对于最高层，由于不存在更高层解码器特征，门控权重由编码器特征和对应的过渡层特征共同生成。
\[
G_i =
P\Big(
\sigma(
\text{Conv}([E_i, T_i])
)
\Big)
\]

门控单元输出两个权重 $G_i = [G_i^{1}, G_i^{2}]$，
其中 \(G_i^{1}\) 作用于 FPN 分支，控制当前层特征参与逐级解码的强度；
\(G_i^{2}\) 作用于 Parallel 分支，用于选择补充残差信息的编码器特征。

% ----------------------------------------------------------------
\subsubsection{FPN 分支}

FPN 分支负责生成主要的显著性预测，其解码过程采用自顶向下融合策略：高层语义特征逐级上采样，并与当前层经过门控筛选的编码器特征融合。

\[
D_i =
\text{Conv}
\Big(
G_i^{1} \cdot T_i + \text{Up}(D_{i+1})
\Big)
\]

在这种机制下，高层特征提供全局语义，低层特征补充空间细节，门控单元则在二者之间起到抑制背景干扰、平衡层级贡献的作用。

% ----------------------------------------------------------------
\subsubsection{Parellel 分支}

逐级解码结构可能导致部分细节信息在自顶向下融合过程中被削弱，GateNet 为此增加了并行分支，将各层经过第二组门控权重筛选后的特征直接汇聚起来，用于补充 FPN 分支遗漏的信息。

\[
F_{\text{cat}} =
\text{Cat}
\Big(
D_1,
\text{Up}(G_1^{2} \cdot T_1),
\text{Up}(G_2^{2} \cdot T_2),
\cdots,
\text{Up}(G_n^{2} \cdot T_n)
\Big)
\]

随后通过卷积生成残差预测，并与 FPN 分支结果相加。
\[
S_{\text{final}}
=
\text{Conv}(F_{\text{cat}}) + D_1
\]

其中 \(D_1\) 表示 FPN 分支得到的初步显著性预测。该残差结构使并行分支不必重新预测完整显著图，而是专注于补充边界、细长结构和局部细节。

从功能上看，FPN 分支更依赖高层语义信息，用于确定显著目标主体；Parellel 分支更强调低层细节信息，用于恢复局部结构。二者通过门控机制形成互补。

\subsection{$\text{F}^3$ Net}

\subsubsection{基本思想}

$\text{F}^3$Net \cite{wei2020f3net}是一种面向显著目标检测的双路径解码网络（图~\ref{fig:f3net}），
其名称来源于三个核心设计理念：特征融合（Fusion）、跨路径反馈（Feedback）与多尺度聚焦（Focus）。
$\text{F}^3$Net 的核心创新点包括：

\begin{itemize}
    \item \textbf{跨层特征交互（CFI）}：通过相互门控机制在高层语义特征与低层空间特征之间建立双向注意力，代替简单拼接或相加。
    \item \textbf{特征融合反馈（FFM）}：以粗粒度路径的解码中间结果为反馈信号，逐级精炼精细路径，使网络在多次迭代中渐进式完善显著图。
    \item \textbf{双路径互补解码}：粗路径（Path 1）自顶向下提取多尺度全局结构，精细路径（Path 2）在粗路径的引导下补充细节。
\end{itemize}

\subsubsection{跨层特征交互模块（CFI）}

跨层特征交互模块（Cross Feature Interaction，CFI）是 $\text{F}^3$Net 的核心融合单元。
与传统 skip connection 直接相加不同，CFI 让两路特征相互生成对方的注意力权重，从而更精确地提取互补信息。

给定高层特征 $F_{\text{hi}} \in \mathbb{R}^{B \times C_h \times H' \times W'}$ 和低层特征 $F_{\text{lo}} \in \mathbb{R}^{B \times C_l \times H \times W}$，
CFI 首先将二者投影到公共通道数 $d$ 并上采样高层特征至低层分辨率：
\[
H = \phi_h(F_{\text{hi}}^{\uparrow}), \quad L = \phi_l(F_{\text{lo}})
\]
随后拼接计算双向门控权重：
\[
A_H = \sigma\!\left(W_h[H, L]\right), \quad A_L = \sigma\!\left(W_l[H, L]\right)
\]
其中 $[\,\cdot\,]$ 表示通道拼接，$\sigma(\cdot)$ 为 Sigmoid，$W_h, W_l$ 均为 $1\times1$ 卷积。
各路特征在接收对方门控增强后进行拼接融合：
\[
H' = H \odot A_H + L, \quad L' = L \odot A_L + H
\]
\[
F_{\text{CFI}} = \text{Conv}_{3\times3}([H', L'])
\]
这种双向交互使高层特征在语义层面选择与低层结构对应的区域，低层特征则在空间层面关注被高层语义激活的显著位置，二者协同提升融合质量。

% ----------------------------------------------------------------
\subsubsection{特征融合反馈模块（FFM）}

特征融合反馈模块（Feature Fusion Feedback Module，FFM）负责将粗路径（Path 1）在同一尺度的中间特征作为先验信号注入精细路径（Path 2），
实现跨路径的渐进式精炼。

给定精细路径当前特征 $X$ 和粗路径反馈特征 $F_{\text{fb}}$，FFM 首先将后者上采样至 $X$ 的分辨率，再拼接融合：
\[
F_{\text{fuse}} = \text{Conv}_{3\times3}([X, F_{\text{fb}}^{\uparrow}])
\]
随后通过通道注意力（SE-style）对融合特征进行自适应重标定：
\[
A_c = \sigma\!\left(\text{FC}_2\!\left(\delta\!\left(\text{FC}_1\!\left(\text{GAP}(F_{\text{fuse}})\right)\right)\right)\right)
\]
最终以残差形式输出：
\[
F_{\text{FFM}} = F_{\text{fuse}} \odot A_c + F_{\text{fuse}}
\]
残差连接保证即使注意力权重接近零时网络也不会丢失原有融合信息，使训练更加稳定。

% ----------------------------------------------------------------
\subsubsection{双路径解码流程}

$\text{F}^3$Net 采用共享骨干网络的双路径解码器。骨干网络依次输出 $C_1, C_2, C_3, C_4$ 四个尺度的特征（分辨率逐渐降低，通道数逐渐增加）。

\textbf{Path 1（粗路径）}从最深层开始，通过三次 CFI 自顶向下完成粗粒度解码：
\[
P_3^{(1)} = \text{CFI}(C_4, C_3), \quad
P_2^{(1)} = \text{CFI}(P_3^{(1)}, C_2), \quad
P_1^{(1)} = \text{CFI}(P_2^{(1)}, C_1)
\]

\textbf{Path 2（精细路径）}在 CFI 的基础上引入 FFM 接收 Path 1 的反馈：
\[
P_3^{(2)} = \text{FFM}\!\left(\text{CFI}(C_4, C_3),\ P_3^{(1)}\right)
\]
\[
P_2^{(2)} = \text{FFM}\!\left(\text{CFI}(P_3^{(2)}, C_2),\ P_2^{(1)}\right)
\]
\[
P_1^{(2)} = \text{FFM}\!\left(\text{CFI}(P_2^{(2)}, C_1),\ P_1^{(1)}\right)
\]

最终主预测图由 Path 2 的最高分辨率特征 $P_1^{(2)}$ 经 $1\times1$ 卷积生成，
辅助预测图由 Path 1 的 $P_1^{(1)}$ 生成，训练时共同优化。

\subsection{PoolNet-DS}

PoolNet-DS 在 PoolNet 主干结构基础上引入多尺度深监督（Deep Supervision）与边缘感知约束。
原始 PoolNet 仅在最终解码特征上生成显著图，训练信号需要经过完整解码路径反向传播到中间层，
容易使中高层融合节点缺少直接监督。为缓解这一问题，我们在三个中间解码阶段分别设置辅助显著性预测头，
并在浅层高分辨率特征上增加边缘预测分支，使网络同时关注区域完整性与边界结构。

设自顶向下解码过程中得到的三个中间特征为 \(M_3,M_2,M_1\)，其中 \(M_1\) 分辨率最高、包含更多局部细节。
PoolNet-DS 对每个中间特征使用独立的多尺度预测头生成辅助显著图：
\[
S_i^{aux}
=
\operatorname{Up}
\left(
H_i(M_i)
\right)
\]
其中 \(H_i(\cdot)\) 表示由两层 \(3\times3\) 卷积和 \(1\times1\) 预测卷积组成的辅助头，
\(\operatorname{Up}(\cdot)\) 表示上采样至输入图像尺寸。
对于浅层特征 \(M_1\)，额外使用边缘分支预测边界图：
\[
E_1
=
\operatorname{Up}
\left(
\operatorname{Conv}_{1\times1}
\left(
\operatorname{ReLU}
\left(
\operatorname{BN}
\left(
\operatorname{Conv}_{3\times3}(M_1)
\right)
\right)
\right)
\right)
\]
训练时，主显著图仍采用 BCE、SSIM 与 Soft IoU 的组合损失；
辅助显著图采用 BCE 与 Soft IoU 约束，边缘分支则以由真实掩码 Sobel 梯度生成的边缘图作为监督信号。
因此，整体损失可写为：
\[
\mathcal{L}
=
\mathcal{L}_{main}
+
\sum_i \lambda_i \mathcal{L}_{aux}(S_i^{aux},G)
+
\lambda_e \mathcal{L}_{edge}(E_1,\nabla G)
\]
其中 \(G\) 表示真实显著性标注，\(\nabla G\) 表示归一化后的 Sobel 边缘图。

% ----------------------------------------------------------------
\subsection{PoolNet-Gate}

PoolNet-Gate 针对 PoolNet 中 skip connection 直接相加可能引入背景噪声的问题，
在每个需要融合浅层特征的 Deep Pooling Layer 中加入双分支门控模块。
该模块并不改变 PoolNet 的整体编码器--解码器框架，而是在高层解码特征与浅层 skip 特征融合之前，
自适应筛选浅层特征中的有效通道和有效空间位置。

具体而言，给定当前解码特征 \(D\) 与同分辨率浅层特征 \(L\)，通道门控分支首先对 \(D\) 做全局平均池化，
再通过两层 \(1\times1\) 卷积生成通道权重：
\[
A_c
=
\sigma
\left(
W_2
\delta
\left(
W_1
\operatorname{GAP}(D)
\right)
\right),
\quad
L_c = A_c \odot L
\]
其中 \(\sigma(\cdot)\) 为 Sigmoid 函数，\(\delta(\cdot)\) 为 ReLU 激活，\(\odot\) 表示逐元素乘法。
空间门控分支则拼接 \(D\) 与 \(L\)，通过卷积预测空间注意力图：
\[
A_s
=
\sigma
\left(
\operatorname{Conv}_{1\times1}
\left(
\delta
\left(
\operatorname{BN}
\left(
\operatorname{Conv}_{3\times3}([D,L])
\right)
\right)
\right)
\right),
\quad
L_s = A_s \odot L
\]
两路门控结果进一步拼接并用卷积融合：
\[
\hat{L}
=
\operatorname{Conv}_{3\times3}([L_c,L_s])
\]
最终，PoolNet-Gate 的融合形式由原始的
\(D+L+I\) 改为 \(D+\hat{L}+I\)，其中 \(I\) 为 PPM 产生的全局引导特征。
这种设计使通道门控负责选择语义相关的特征维度，空间门控负责抑制非显著区域响应，
二者互补地提升了浅层细节特征的可靠性。

% ----------------------------------------------------------------
\subsection{PoolNet-ASPP}

PoolNet-ASPP 的动机是增强深层解码阶段的多尺度上下文建模能力。
原始 PoolNet 虽然通过 PPM 提供全局语义引导，并在 Deep Pooling Layer 内使用多尺度平均池化聚合上下文，
但在最深层解码时仍主要依赖局部卷积完成特征变换。
显著目标检测中，目标尺度变化较大，复杂背景中的局部高响应区域也容易被误判为显著目标；
因此，我们在最深的两个解码单元中用 ASPP 模块替换原有 FAM 结构，以增加不同感受野下的语义判别能力。

设深层输入特征为 \(X\)，ASPP 模块包含 \(1\times1\) 卷积分支、多个空洞卷积分支以及全局平均池化分支：
\[
Y
=
\operatorname{Proj}
\left(
\left[
\operatorname{Conv}_{1\times1}(X),
\operatorname{Conv}_{3\times3}^{r=2}(X),
\operatorname{Conv}_{3\times3}^{r=4}(X),
\operatorname{Conv}_{3\times3}^{r=8}(X),
\operatorname{Up}(\operatorname{GAP}(X))
\right]
\right)
\]
其中 \(r\) 表示空洞卷积的 dilation rate，\(\operatorname{Proj}(\cdot)\) 为 \(1\times1\) 投影卷积。
在解码流程中，前两个从深到浅的融合节点采用
\[
D_i
=
\operatorname{ASPP}
\left(
\operatorname{Up}(D_{i+1})
\right)
+
L_i
+
I_i
\]
而浅层两个节点仍保留 PoolNet 原有的 FAM 结构。
这样的分层设计使深层特征更关注尺度鲁棒的语义定位，浅层特征则继续负责边界和局部结构恢复，
避免在所有层统一使用 ASPP 带来的计算冗余。

% ----------------------------------------------------------------
\subsection{PoolNet-CFM}

PoolNet-CFM 将原始 PoolNet 中的直接加法融合替换为跨层特征交互模块（Cross Feature Module，CFM）。
直接相加默认高层解码特征与浅层 skip 特征已经在语义上对齐，
但二者分别偏向语义定位与空间细节，简单相加可能导致响应混杂。
因此，PoolNet-CFM 借鉴 \(\text{F}^3\) Net 中的交互思想，
先分别对两路特征进行卷积变换，再通过乘法建模二者的一致响应。

给定解码特征 \(D\) 和浅层特征 \(L\)，CFM 首先得到两路二阶特征：
\[
D_2 = f_d^{(2)}(f_d^{(1)}(D)),
\quad
L_2 = f_l^{(2)}(f_l^{(1)}(L))
\]
随后使用逐元素乘法获得交互特征：
\[
F_{\times} = D_2 \odot L_2
\]
该乘法项相当于软一致性约束，也就是只有在高层语义与浅层细节共同响应的位置，
融合特征才会被显著增强。之后，交互结果分别回注到两条分支并进行残差精炼：
\[
\hat{D}
=
f_d^{(4)}
\left(
f_d^{(3)}(F_{\times}) + f_d^{(1)}(D)
\right),
\quad
\hat{L}
=
f_l^{(4)}
\left(
f_l^{(3)}(F_{\times}) + f_l^{(1)}(L)
\right)
\]
最终输出为
\[
F_{cfm} = \hat{D} + \hat{L}
\]
在 PoolNet-CFM 的每个融合节点中，解码特征先经过 \(3\times3\) 卷积调整通道，
再与 skip 特征送入 CFM，最后叠加全局引导特征 \(I\) 并进入 FAM：
\[
D_i
=
\operatorname{FAM}
\left(
\operatorname{CFM}(D_{i+1}^{\uparrow},L_i)+I_i
\right)
\]
该结构使融合过程从无条件相加升级为语义与细节一致性驱动的交互融合。

% ----------------------------------------------------------------
\subsection{PoolNet-Gate-CFM}

PoolNet-Gate-CFM 进一步将门控筛选与 CFM 交互融合结合起来。
PoolNet-Gate 可以在融合前抑制浅层背景噪声，PoolNet-CFM 则能够增强高层语义与浅层细节的一致响应，
可以形成互补。

在该模型中，每个需要 skip 融合的解码单元首先利用双分支门控得到净化后的浅层特征：
\[
\hat{L}_i
=
\operatorname{DualGate}(D_i,L_i)
\]
随后将当前解码特征与 \(\hat{L}_i\) 输入 CFM：
\[
D_i'
=
\operatorname{CFM}(D_i,\hat{L}_i)+I_i
\]
最后再通过 FAM 进行多尺度池化增强：
\[
D_i
=
\operatorname{FAM}(D_i')
\]
其中，门控模块减少 skip 特征中与显著目标无关的通道和空间响应，
CFM 则在较干净的浅层特征基础上建模跨层乘法关系。

% ----------------------------------------------------------------
\subsection{PoolNet-ASPP-Gate}

PoolNet-ASPP-Gate 采用分层混合解码策略：在深层两个解码单元使用 ASPP，
在浅层两个解码单元使用双分支 Gate。
这一设计基于显著目标检测中不同层级特征的功能差异，
深层特征分辨率较低但语义信息强，更适合通过 ASPP 扩大感受野、建模尺度变化；
而浅层特征分辨率较高但背景噪声较多，更适合通过门控机制筛选边界与局部纹理信息。

令 \(i=0,1\) 表示较深的两个解码阶段，\(i=2,3\) 表示较浅的两个解码阶段，
PoolNet-ASPP-Gate 的解码单元可概括为：
\[
D_i =
\begin{cases}
\operatorname{ASPP}(D_{i+1}^{\uparrow}) + L_i + I_i, & i \in \{0,1\}\\
\operatorname{FAM}(D_{i+1}^{\uparrow} + \operatorname{DualGate}(D_{i+1}^{\uparrow},L_i) + I_i), & i = 2\\
\operatorname{FAM}(D_{i+1}), & i = 3
\end{cases}
\]
相比于所有层统一引入同一种模块，这种方式更符合 PoolNet 自顶向下解码的语义分工。
其中深层负责确定显著目标的大致位置与尺度，浅层负责恢复边缘和细节。

% ----------------------------------------------------------------
\subsection{PoolNet-ASPP-CFM}

PoolNet-ASPP-CFM 关注深层语义增强与跨层交互融合的联合建模，
在最深两个解码阶段同时使用 ASPP 与 CFM。
ASPP 先增强当前解码特征的多尺度语义表达，CFM 再将其与对应 skip 特征进行乘法交互。

对于深层融合节点，其计算过程为：
\[
\tilde{D}_i
=
\operatorname{ASPP}(D_{i+1}^{\uparrow})
\]
\[
D_i
=
\operatorname{CFM}(\tilde{D}_i,L_i)+I_i
\]
浅层节点则保留原始 PoolNet 的 FAM 结构：
\[
D_i
=
\operatorname{FAM}(D_{i+1}^{\uparrow}+L_i+I_i)
\]
这种设计的核心假设是，深层语义特征经过 ASPP 之后具有更强的尺度鲁棒性，
再与 skip 特征进行乘法交互时，可以更准确地选择与显著目标相关的空间细节。

% ----------------------------------------------------------------
\subsection{PoolNet-CFM(Enhanced)}

PoolNet-CFM(Enhanced) 是在 PoolNet-CFM 基础上得到的增强版本。
该模型保留 CFM 作为主要跨层融合方式，同时对解码尾部加入前景门控（Foreground Gate），
用于根据深层融合结果对最终高分辨率特征进行再加权。

具体而言，模型首先执行与 PoolNet-CFM 相同的自顶向下解码，并在第一个融合节点得到深层引导特征 \(G_f\)：
\[
G_f = D_0
\]
随后经过后续解码单元得到最终特征 \(D_{final}\)。
前景门控以 \(G_f\) 为 guide 预测单通道前景权重图：
\[
A_f
=
\sigma
\left(
\operatorname{Conv}_{1\times1}
\left(
\delta
\left(
\operatorname{BN}
\left(
\operatorname{Conv}_{3\times3}(G_f)
\right)
\right)
\right)
\right)
\]
最终特征按如下方式进行门控：
\[
\hat{D}_{final}
=
D_{final}
\odot
\left(
(1-\alpha)+\alpha A_f
\right)
\]

从模型设计上看，Enhanced 版本将 CFM 的跨层一致性建模与深层前景先验结合起来，
既利用浅层细节恢复边界，又利用深层语义约束最终响应区域。

% ----------------------------------------------------------------
\subsection{$\text{F}^3$Net-CBAM}

$\text{F}^3$Net-CBAM 在 $\text{F}^3$Net 双路径解码器的每个 CFI 和 FFM 输出后插入卷积块注意力模块（Convolutional Block Attention Module，CBAM），
对中间特征进行通道与空间两级精炼，再送入下一解码阶段。

CBAM 由串联的通道注意力和空间注意力组成。
对于输入特征 $F \in \mathbb{R}^{B \times C \times H \times W}$，通道注意力分支分别经过全局平均池化与最大池化后共享 MLP 并相加：
\[
A_c = \sigma\!\left(\text{MLP}(\text{AvgPool}(F)) + \text{MLP}(\text{MaxPool}(F))\right)
\]
\[
F' = F \odot A_c
\]
空间注意力分支沿通道维度分别取均值和最大值后拼接，经 $7\times7$ 卷积生成二维注意力图：
\[
A_s = \sigma\!\left(\text{Conv}_{7\times7}\!\left([\bar{F}', \hat{F}']\right)\right)
\]
\[
\tilde{F} = F' \odot A_s
\]
其中 $\bar{F}' = \text{mean}(F', \text{dim}=1)$，$\hat{F}' = \text{max}(F', \text{dim}=1)$。

在 $\text{F}^3$Net-CBAM 中，每个解码阶段的 CBAM 按如下方式嵌入：
\[
P_i^{(1)} = \text{CBAM}_i^{(1)}\!\left(\text{CFI}(P_{i+1}^{(1)}, C_i)\right)
\]
\[
P_i^{(2)} = \text{CBAM}_i^{(2)}\!\left(\text{FFM}\!\left(\text{CFI}(P_{i+1}^{(2)}, C_i),\ P_i^{(1)}\right)\right)
\]
CBAM 使每阶段特征在送入下一层 CFI 之前先完成通道维度的语义聚焦和空间维度的前景定位，
抑制非显著区域的无效响应，从而提高跨层交互的信噪比。

% ----------------------------------------------------------------
\subsection{$\text{F}^3$Net-ASPP}

$\text{F}^3$Net-ASPP 的动机是增强 CFI 模块在聚合高层特征时的多尺度感知能力。
原始 $\text{F}^3$Net 中，CFI 通过 $1\times1$ 卷积将高层特征投影到公共通道，
这种单尺度投影在目标尺度变化较大时难以同时建模全局语义和局部结构。

为此，$\text{F}^3$Net-ASPP 将 CFI 中的高层特征投影替换为轻量级 ASPP（CFI\_ASPP），
在进行跨层交互之前先用多分支空洞卷积扩大高层特征的感受野：
\[
H = \text{ASPP}(F_{\text{hi}}^{\uparrow}), \quad L = \phi_l(F_{\text{lo}})
\]
ASPP 模块包含五条并行分支——$1\times1$ 卷积、空洞率 $r\in\{2,4,6\}$ 的 $3\times3$ 空洞卷积及全局平均池化分支——输出拼接后经 $1\times1$ 投影压缩：
\[
\text{ASPP}(X) = \text{Proj}\!\left(\left[b_0(X), b_1(X), b_2(X), b_3(X), \text{Up}(b_4(X))\right]\right)
\]
CFI\_ASPP 的后续交互流程与原始 CFI 保持一致，通过双向门控融合高层与低层特征。

这种设计使高层特征在与低层进行跨层交互前已具备多尺度语义感知，
从而在注意力权重计算时对大尺度目标和小尺度细节均能有效响应。

% ----------------------------------------------------------------
\subsection{GateNet-CBAM}

GateNet-CBAM 在 GateNet 的深度编码模块（DEM）输出与门控单元之间插入 CBAM 对过渡特征进行精炼。

GateNet 原始流程中，DEM 将骨干网络各层特征 $E_i$ 压缩为过渡特征 $T_i$，
再送入门控单元参与 FPN 解码。然而，$T_i$ 直接来自卷积压缩，可能保留了背景噪声，
影响后续门控权重的质量。

GateNet-CBAM 在每个 DEM 之后增加 CBAM 精炼：
\[
T_i = \text{CBAM}_i\!\left(\text{DEM}_i(E_i)\right), \quad i \in \{1,2,3,4\}
\]
随后门控单元、FPN 分支与并行分支的流程与原始 GateNet 完全相同：
\[
G_i = \text{GateModule}\!\left([E_i, D_{i+1}]\right)
\]
\[
D_i = \text{FPN}_i\!\left(G_i^{(1)} \odot T_i\right) + \text{Up}(D_{i+1})
\]
CBAM 的通道注意力使 $T_i$ 在语义维度上聚焦于显著相关的特征通道，
空间注意力进一步抑制 $T_i$ 中的背景区域响应，
使后续门控单元在更纯净的过渡特征上生成权重，从而减少背景干扰传入 FPN 主干的可能性。

% ----------------------------------------------------------------
\subsection{GateNet-DS}

GateNet-DS 针对 GateNet 仅在最终输出处计算损失、中间解码层缺少直接监督信号的问题，
在 FPN 分支的第三（$D_3$）和第二（$D_2$）解码阶段分别添加辅助显著性预测头，实现深监督。

中间解码特征 $D_3, D_2$ 分别经过 $1\times1$ 卷积生成辅助预测图，
并上采样至输入分辨率：
\[
S_3^{\text{aux}} = \text{Up}\!\left(\text{Conv}_{1\times1}(D_3)\right), \quad
S_2^{\text{aux}} = \text{Up}\!\left(\text{Conv}_{1\times1}(D_2)\right)
\]
训练时，辅助预测图与主预测图 $S_{\text{final}}$ 共同参与损失计算：
\[
\mathcal{L} = \mathcal{L}_{\text{main}}(S_{\text{final}}, G)
+ \lambda_1 \mathcal{L}_{\text{aux}}(S_3^{\text{aux}}, G)
+ \lambda_2 \mathcal{L}_{\text{aux}}(S_2^{\text{aux}}, G)
\]
其中 $G$ 为真实标注，$\lambda_1, \lambda_2$ 为辅助损失权重（均设为 0.4），辅助损失 $\mathcal{L}_{\text{aux}}$ 采用 BCE 与 Soft IoU 的组合。
推理阶段辅助头被移除，不引入额外计算开销。

深监督的引入使梯度可以直接传播至 $D_3, D_2$ 特征提取路径，
避免因梯度经由过长的解码链回传而导致中间层收敛不充分，
从而提升网络对中等分辨率特征的显著目标定位能力。

%----------------------------------------------------------------
\subsubsection{MAE}

MAE（Mean Absolute Error）表示平均绝对误差，用于衡量预测显著图与真实标注图之间的像素级差异。其定义为：
\begin{equation*}
  MAE=\frac{1}{W\times H}\sum_{x=1}^W\sum_{y=1}^H|S(x,y)-G(x,y)|
\end{equation*}

其中，$S$ 表示预测显著图，$G$ 表示真实标注图。MAE 直接在连续显著图上进行计算，
不依赖于二值化阈值。因此，MAE 能够反映预测图与真实标注图在整体像素层面的接近程
度。MAE 数值越小，说明模型生成的显著图与真实标注越接近，预测误差越低。

%----------------------------------------------------------------
\subsubsection{F-measure}

F-measure 是 Precision 和 Recall 的加权调和平均，用于综合衡量模型的准确性和完整性。其定义为：
\begin{equation*}
  F_{\beta}=\frac{(1+\beta^2)\cdot Precision \cdot Recall}{\beta^2\cdot Precision + Recall}
\end{equation*}

其中，$\beta$ 用于调节 Precision 和 Recall 的相对重要性，通常设置 $\beta^2=0.3$。
F-measure 数值越大，说明模型在显著目标区域检测的准确性和完整性方面表现越好。

%----------------------------------------------------------------
\subsubsection{Weighted F-measure}

Weighted F-measure（$F_{\beta}^{w}$）是传统 F-score 的加权扩展形式，用于更加准确地评价显著性检测结果。普通 F-score 主要依据像素级 Precision 和 Recall 计算模型性能，但对预测误差的空间位置和结构影响考虑不足。为此，$F_{\beta}^{w}$ 引入加权机制，对不同区域的预测误差赋予不同权重，从而能够更好地反映显著目标的区域完整性、边界质量以及背景抑制效果。
其定义为：
\begin{equation*}
  F_{\beta}^{w}
  =
  \frac{(1+\beta^2)\cdot Precision^{w}\cdot Recall^{w}}
  {\beta^2\cdot Precision^{w}+Recall^{w}}
\end{equation*}

其中，$Precision^{w}$ 表示加权准确率，$Recall^{w}$ 表示加权召回率，$\beta$ 用于调节二者的相对重要性。$F_{\beta}^{w}$ 数值越大，说明预测显著图与真实标注图在像素分布和空间结构上越一致，模型的显著性检测性能越好。

\subsubsection{S-measure}

S-measure（Structural-measure，$S_m$）是一种用于评价显著性检测模型预测结果与真实标注之间结构一致性的指标。S-measure 同时考虑区域一致性（Region-aware）与对象一致性（Object-aware）两方面信息，
从而能够更全面地评估预测显著图在整体形状、前景分布以及边界结构上的质量。其定义为：
\begin{equation*}
  S_m = \alpha \cdot S_o + (1-\alpha) \cdot S_r
\end{equation*}

其中，$S_o$ 表示对象级一致性（Object-aware similarity），度量预测显著图与真实标注在前景整体分布上的相似程度；$S_r$ 表示区域级一致性（Region-aware similarity），衡量局部区域的结构和边界对齐效果；$\alpha$ 是调节二者相对重要性的平衡系数，通常取 0.5。
S-measure 越大，说明预测显著图在结构和形状上越接近真实标注，能够更好地反映显著目标的完整性、边界精确度以及前景与背景的分离效果。

\subsection{实验细节}

本项目基于 PyTorch 深度学习框架实现，使用 NVIDIA RTX 4090 GPU 完成模型训练与测试。
为保证不同模型之间的公平比较，各实验均采用 ImageNet 预训练的 ResNet18 或 ResNet-34 作为骨干网络。
实验默认使用 DUTS-TR 作为训练集，并在 ECSSD 数据集上进行测试与性能评估。

在数据预处理阶段，图像被缩放为 $320 \times 320$ 作为网络输入。
训练阶段采用随机水平翻转进行数据增强，同时使用 ImageNet 预训练模型的统计量对输入图像进行归一化。
显著性标注图的二值化阈值设置为 $0.5$。

优化器采用 AdamW，权重衰减系数设为 $5\times10^{-4}$。
为避免预训练骨干网络参数发生过大扰动，实验对骨干网络与新建层采用不同学习率：
骨干网络学习率设为 $3\times10^{-5}$，其余新建参数学习率设为 $2\times10^{-4}$。
所有模型均训练 25 个 epoch。学习率调度采用 MultiStep 策略，
在第 15 个和第 22 个 epoch 将当前学习率衰减为原来的 $\frac{1}{10}$。

训练损失由显著性检测中常用的像素级分类损失与结构约束损失共同组成。
对于主输出，采用 BCE、SSIM Loss 与 Soft IoU Loss 的加权组合：
\[
\mathcal{L}_{main}
=
\mathcal{L}_{BCE}
+
\mathcal{L}_{SSIM}
+
\left(1-\operatorname{IoU}_{soft}\right)
\]
其中 Soft IoU 定义为
\[
\operatorname{IoU}_{soft}
=
\frac{\sum_i p_i g_i}
{\sum_i (p_i + g_i - p_i g_i) + \epsilon}
\]
$p_i$ 表示预测显著图在第 $i$ 个像素处的概率值，$g_i$ 表示对应的真实标注值，
$\epsilon$ 为防止除零的极小常数。
