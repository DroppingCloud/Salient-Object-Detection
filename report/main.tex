\section{摘要}

本报告围绕显著性目标检测任务展开，以 ResNet-18 \cite{he2015resnet} 为骨干网络，
系统研究了基于深度学习的显著图预测方法。

\bigskip
\noindent\textbf{关键词：}显著性目标检测；ResNet-18；编解码器；注意力机制；多尺度融合

%----------------------------------------------------------------
\section{引言}

显著性目标检测（Salient Object Detection, SOD）旨在从自然图像中定位并分割出最吸引人类视觉注意力的前景区域，
输出与输入图像等尺寸的二值或软性预测掩码（mask）。该任务在图像检索、视频理解、自动驾驶等领域具有广泛的应用价值。

传统方法主要依赖于在手工设计的特征下进行，难以适应复杂图像场景和结构，泛化能力有限。
随着深度学习的发展，基于卷积神经网络的方法在语义特征提取上取得了显著进步，成为当前的主流范式。

我们的贡献总结如下：

\begin{enumerate}
  \item 实现了两种基于 ResNet-18 的基线方法（FCN 解码器与 U-Net 跳跃连接解码器）并进行对比分析；
  \item 提出了融合边缘增强分支、通道注意力与多尺度特征融合的改进网络（ResSOD-Improved）；
  \item 设计了结合二值交叉熵（BCE）与结构相似性（SSIM）的混合损失函数；
  \item 在 DUTS 数据集上进行了系统的消融实验，定量验证了各模块的贡献。
\end{enumerate}

%----------------------------------------------------------------
\section{相关工作}

% ----------------------------------------------------------------
\subsection{基于 CNN 的显著性检测方法}

早期深度学习方法将 VGG、ResNet 等分类网络改造为全卷积网络（FCN）\cite{he2015resnet}，
直接对特征图进行上采样以输出显著图。
此类方法结构简单，但由于编码过程中空间信息丢失，边界预测往往模糊。

% ----------------------------------------------------------------
\subsection{编解码器结构}

U-Net \cite{ronneberger2015unet} 引入跳跃连接，将编码器各层的高分辨率特征图与解码器对应层融合，
显著提升了边界定位精度。该思想被广泛应用于显著性检测任务，
衍生出 PoolNet、F3Net、U$^2$-Net 等一系列方法。

% ----------------------------------------------------------------
\subsection{注意力机制}

通道注意力（SENet 风格）和空间注意力（CBAM）被引入 SOD 模型以增强对显著区域的聚焦能力。
注意力模块通过重新校正特征权重，使网络在解码阶段更关注语义相关区域。

% ----------------------------------------------------------------
\subsection{多尺度特征融合}

不同尺度的特征图分别捕捉粗粒度语义信息和细粒度空间信息。
特征金字塔网络（FPN）、渐进融合等策略被用于融合多层特征，改善对大尺寸目标和小尺寸目标的检测能力。

%----------------------------------------------------------------
\section{方法}

% ----------------------------------------------------------------
\subsection{Baseline 1：PoolNet}

以预训练 ResNet-18 作为编码器，提取最后一层特征图（步长 32，通道数 512），
经两次双线性上采样（$\times 16$、$\times 2$）还原至输入分辨率，
最后通过 $1\times1$ 卷积与 Sigmoid 激活输出显著图。
损失函数采用像素级二值交叉熵（BCE）。

该方法结构最为简洁，可作为验证任务可行性的最低基准。

% ----------------------------------------------------------------
\subsection{Baseline 2：$\text{U}^2$ Net}

保留 ResNet-18 编码器，在解码器中引入 U-Net 风格的跳跃连接：
将编码器 Stage 1\textasciitilde4 的特征图（分辨率分别为 $128^2$、$64^2$、$32^2$、$16^2$）
与解码器对应层的上采样特征拼接后经卷积融合，最终输出与输入等尺寸的显著图 \cite{ronneberger2015unet}。
该方法在恢复空间细节方面优于 Baseline 一。

% ----------------------------------------------------------------
\subsection{Baseline 3：$\text{F}^3$ Net}

在 ResNet18-UNet 基础上引入以下三个模块：

% ----------------------------------------------------------------
\subsubsection{边缘增强分支（Edge Branch）}

在解码器最高分辨率层附加一路边缘检测分支，以 Canny 算法提取的边缘图作为弱监督信号，
驱动网络学习目标边界的精细结构。边缘分支输出 $\hat{\mathbf{E}} \in [0,1]^{H \times W}$，
与主分支显著图融合后提升边界锐度。

\subsubsection{通道注意力模块（Channel Attention）}

在每个解码器融合块后引入 Squeeze-and-Excitation（SE）模块：
对特征图执行全局平均池化，经两层全连接网络生成通道权重向量，
对特征图各通道进行重加权，以增强对显著语义特征的响应。

\subsubsection{多尺度融合模块（Multi-Scale Fusion）}

引入特征金字塔聚合策略，将编码器 Stage 2\textasciitilde4 的特征图通过 $1\times1$ 卷积对齐通道后，
双线性上采样至 $64\times64$ 统一分辨率，逐元素加和融合，
再送入解码器，缓解单一尺度特征对大/小目标检测的局限。

\subsection{网络整体结构}

\begin{figure}[H]
  \centering
  \fbox{\rule{0pt}{3cm}\rule{10cm}{0pt}}
  \caption{ResSOD-Improved 整体网络结构示意图}
  \label{fig:arch}
\end{figure}

整体流程如下：
\begin{enumerate}
  \item 输入图像经 ResNet-18 编码器提取多尺度特征 $\{f_1, f_2, f_3, f_4\}$；
  \item 多尺度融合模块聚合 $\{f_2, f_3, f_4\}$ 生成融合特征 $f_\text{ms}$；
  \item 解码器逐层上采样，每层接收跳跃连接特征与通道注意力加权，输出特征图；
  \item 边缘分支在最高分辨率层提取边缘信息辅助训练；
  \item 最终 $1\times1$ 卷积 + Sigmoid 输出显著图 $\hat{\mathbf{S}}$。
\end{enumerate}

\subsection{损失函数设计}

混合损失函数由 BCE 损失与 SSIM 损失加权组合：
\begin{equation}
  \mathcal{L} = \lambda_{\text{bce}}\,\mathcal{L}_{\text{BCE}}
              + \lambda_{\text{ssim}}\,\mathcal{L}_{\text{SSIM}}
              + \lambda_{\text{edge}}\,\mathcal{L}_{\text{Edge}}
  \label{eq:loss}
\end{equation}

其中：
\begin{itemize}
  \item $\mathcal{L}_{\text{BCE}} = -\frac{1}{HW}\sum_{i}[G_i\log\hat{S}_i + (1-G_i)\log(1-\hat{S}_i)]$，
        像素级二值交叉熵，保证逐像素分类精度；
  \item $\mathcal{L}_{\text{SSIM}} = 1 - \text{SSIM}(\hat{\mathbf{S}}, \mathbf{G})$，
        结构相似性损失，关注整体结构与区域一致性；
  \item $\mathcal{L}_{\text{Edge}} = \text{BCE}(\hat{\mathbf{E}}, \mathbf{E}_\text{gt})$，
        边缘分支监督损失，$\mathbf{E}_\text{gt}$ 为 Canny 提取的边缘标签。
\end{itemize}

实验中默认设置 $\lambda_{\text{bce}}=1.0$，$\lambda_{\text{ssim}}=0.5$，$\lambda_{\text{edge}}=0.3$。

\subsection{评价指标}

%----------------------------------------------------------------
\subsubsection{Precision-Recall(PR)}

Precision-Recall 曲线用于衡量模型在不同阈值下的检测性能。其中，Precision 表示模型预测
为显著目标的像素中实际为显著目标的比例，反映预测结果的准确性；Recall 表示真实显著目标像素
中被模型正确检测出的比例，反映模型对目标区域的覆盖能力。二者定义如下：
\begin{align*}
  Precision&=\frac{|M \cap G|}{|M|}\\
  Recall&=\frac{|M \cap G|}{|G|}\\
\end{align*}

其中，$M$ 表示预测显著图二值化结果，$G$ 表示真实标注图。在实际评价中，通过改变二值化阈值，
可以得到多组 Precision 和 Recall 值，并绘制 Precision-Recall 曲线。
通常情况下，PR 曲线越接近右上角，说明模型在保持较高召回率的同时仍能维持较高精确率，检测效果越好。

%----------------------------------------------------------------
\subsubsection{F-measure}

F-measure 是 Precision 和 Recall 的加权调和平均，用于综合衡量模型的准确性和完整性。其定义为：
\begin{equation*}
  F_{\beta}=\frac{(1+\beta^2)\cdot Precision \cdot Recall}{\beta^2\cdot Precision + Recall}
\end{equation*}

其中，$\beta$ 用于调节 Precision 和 Recall 的相对重要性，通常设置 $\beta^2=0.3$。
F-measure 数值越大，说明模型在显著目标区域检测的准确性和完整性方面表现越好。

%----------------------------------------------------------------
\subsubsection{Weighted F-measure}

Weighted F-measure（$F_{\beta}^{w}$）是传统 F-score 的加权扩展形式，用于更加准确地评价显著性检测结果。普通 F-score 主要依据像素级 Precision 和 Recall 计算模型性能，但对预测误差的空间位置和结构影响考虑不足。为此，$F_{\beta}^{w}$ 引入加权机制，对不同区域的预测误差赋予不同权重，从而能够更好地反映显著目标的区域完整性、边界质量以及背景抑制效果。
其定义为：
\begin{equation*}
  F_{\beta}^{w}
  =
  \frac{(1+\beta^2)\cdot Precision^{w}\cdot Recall^{w}}
  {\beta^2\cdot Precision^{w}+Recall^{w}}
\end{equation*}

其中，$Precision^{w}$ 表示加权准确率，$Recall^{w}$ 表示加权召回率，$\beta$ 用于调节二者的相对重要性。$F_{\beta}^{w}$ 数值越大，说明预测显著图与真实标注图在像素分布和空间结构上越一致，模型的显著性检测性能越好。

%----------------------------------------------------------------
\subsubsection{MAE}

MAE（Mean Absolute Error）表示平均绝对误差，用于衡量预测显著图与真实标注图之间的像素级差异。其定义为：
\begin{equation*}
  MAE=\frac{1}{W\times H}\sum_{x=1}^W\sum_{y=1}^H|S(x,y)-G(x,y)|
\end{equation*}

其中，$S$ 表示预测显著图，$G$ 表示真实标注图。MAE 直接在连续显著图上进行计算，
不依赖于二值化阈值。因此，MAE 能够反映预测图与真实标注图在整体像素层面的接近程
度。MAE 数值越小，说明模型生成的显著图与真实标注越接近，预测误差越低。

%----------------------------------------------------------------
\subsubsection{E-measure}

E-measure（Enhanced-alignment Measure）是一种综合考虑全局统计信息与局部像素匹配关系的显著性检测评价指标，用于衡量预测显著图与真实标注图之间的整体对齐程度和结构一致性。其定义为：
\begin{equation*}
  E_{\phi}=\frac{1}{W\times H}\sum_{x=1}^{W}\sum_{y=1}^{H}\phi(x,y)
\end{equation*}

其中，$\phi(x,y)$ 表示增强对齐矩阵在像素位置 $(x,y)$ 处的取值，$W$ 和 $H$ 分别表示图像的宽度和高度。E-measure 在评价过程中不仅关注预测结果与真实标注之间的像素级差异，还引入了全局均值信息来刻画二者在整体区域分布上的一致性。因此，相比仅基于像素分类结果的评价指标，E-measure 能够更全面地反映显著目标区域的完整性、前景与背景的区分能力以及预测结果的结构质量。
E-measure 越大表示预测显著图与真实标注图之间的一致性越高，模型的显著性检测效果越好。
